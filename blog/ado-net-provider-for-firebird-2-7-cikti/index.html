ADO.NET provider for Firebird 2.7 çıktı

Dec 19, 2011
Firebird .NET

Birçok projemde Firebird kullandığımı zaten bloguma bir göz atan herkes rahatlıkla anlayabilir. Embed özelliği ile projelerime gömebilmem, sadece birkaç mb alan kaplamasıyla gerçekten küçük kurulum dosyaları oluşturabilmem tüm bunların yanında .net ile Entity Framework desteklemesi benim için çok güzel artılar. Her ne kadar Microsoft SQL Server Compact Edition yayınlamış olsa da, özellikleri gerçek bir projenin ihtiyaçlarını karşılamaktan uzak. Bakalım belki SQL Server 2012 sürümü, SQL Server ürününe kanımın ısınmasını sağlayabilir.

Bu sürüm ile gelen değişikler :

  • Giderilen bir takım hatalar
  • Logging konusunda iyileştirmeler

app.config dosyasına :

<system.diagnostics>
    <sources>
        <source name="FirebirdSql.Data.FirebirdClient">
            <listeners>
                <clear />
                <add name="console" type="System.Diagnostics.ConsoleTraceListener">
            </add></listeners>
        </source>
     </sources>
</system.diagnostics>
